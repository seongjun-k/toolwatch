# 선행 사례 · 레퍼런스 정리

공구 반납 감시대 프로젝트 관련 기존 사례 조사 (2026-07-14). 완전히 동일한 조합(YOLO 재고 감시 + RFID 대여자 귀속 + 미확인 반출 경고)은 발견되지 않음 — 조각별 선행 사례만 존재.

## 1. 비전 기반 공구 감시 (가장 가까운 레퍼런스)

### Edge Impulse — Workplace Organizer (Jetson Nano)
https://docs.edgeimpulse.com/projects/expert-network/workplace-organizer-nvidia-jetson-nano

- 공구 거치대를 카메라로 감시해 미반납 공구를 감지하는 프로젝트. 우리와 문제 정의가 동일 ("교대 근무자가 공구를 제자리에 안 돌려놓는 문제")
- 접근법 두 가지를 비교: ① 공구 종류를 학습시켜 등록 개수와 대조 → 부족하면 경고 (우리 방식과 동일), ② 이상탐지로 "뭔가 빠졌음"만 감지
- **활용**: 우리 판정 알고리즘(개수 대조)이 검증된 접근이라는 근거. 발표 서론·관련 연구에 인용

### YOLOv5 실시간 공구 검출 논문 (ScienceDirect, 2023)
https://www.sciencedirect.com/science/article/pii/S2213846323001190

- 제조 환경에서 YOLOv5로 공구 종류·위치를 실시간 검출. 재고 관리·조립 라인 공구 검증에 활용
- **활용**: 학술 근거. "공구 검출에 YOLO가 적합하다"는 인용용

### Roboflow — Tool Detection 공개 데이터셋
https://universe.roboflow.com/object-detection-yolo-c8gsd/tool-detection-0xiup

- 공구 라벨링 데이터가 이미 공개돼 있음
- **활용**: 1주차 데이터 수집 부담 절감 — 우리가 촬영한 데이터에 섞어서 학습(사전학습/증강용). 단 우리 설치 환경 사진이 주가 되어야 정확도가 나옴

## 2. RFID 대여 귀속

### Etsy rfid-checkout (GitHub)
https://github.com/etsy/rfid-checkout

- 라즈베리파이 + RFID 기반 대여/반납 시스템. 사람 태그 → 60초 안에 물건 태그 → 그 사람에게 귀속
- **활용**: 우리 "RFID 태그 → 30초 세션 → 반출 공구 귀속" 로직과 사실상 동일한 검증된 패턴. 세션 타임아웃 설계 참고

### RC522 재고 관리 프로토타입 (Raspberry Pi 포럼)
https://forums.raspberrypi.com/viewtopic.php?t=268169

- RC522 모듈로 재고 관리 구현 사례. 연동 시 실전 이슈(SPI 설정 등) 참고

### RFID 출결 시스템 튜토리얼 (Pi My Life Up)
https://pimylifeup.com/raspberry-pi-rfid-attendance-system/

- RC522 배선·파이썬 라이브러리 사용법이 단계별로 정리된 튜토리얼
- **활용**: 3주차 RFID 구현 시 배선/코드 1차 참고 자료

## 3. 라즈베리파이 YOLO 추론 (향후 확장용 — Pi 단독 추론 시)

### Ultralytics YOLO on Raspberry Pi (공식)
https://www.raspberrypi.com/news/object-detection-with-ultralytics-yolo26-on-raspberry-pi/

### YOLO11 on Raspberry Pi 최적화 (LearnOpenCV)
https://learnopencv.com/yolo11-on-raspberry-pi/

- Pi 4에서 YOLOv8 tiny 계열 약 1 FPS 수준. NCNN 변환 등 최적화 기법 정리
- **활용**: 우리는 노트북 추론이라 당장 불필요. 4주차 여유 시 "Pi 단독 추론" 부록 시도할 때 참고

## 4. 상용 제품 (시장성 근거)

### henchmanTRAK kabTRAK
https://henchmantrak.com/kabtrak

- 상용 전자식 공구 캐비닛. 전 공구 반납 시 초록, 미반납 시 빨강 표시. 항공정비 등 실수요 현장에서 사용
- **활용**: "실수요가 있는 문제"라는 근거 + 우리 프로젝트는 저비용(카메라 1대 + RC522) 대안이라는 포지셔닝

## 우리 프로젝트의 차별점 (발표용 정리)

기존 사례는 비전 감시(Edge Impulse)와 RFID 귀속(Etsy)이 **각각 따로** 존재. 이 둘을 결합해:

1. 비전으로 "무엇이 언제" 반출/반납됐는지 자동 파악
2. RFID로 "누가" 가져갔는지 귀속
3. 태그 없는 반출은 즉시 경고 — 도난 감시까지 확장
4. 상용 캐비닛(kabTRAK) 대비 저비용 구성

→ "저비용 통합 공구 관리·도난 감시 시스템"으로 포지셔닝.
